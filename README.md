# CardioVision Analytics

Healthcare Analytics & Cardiac Risk Intelligence Dashboard developed using Power BI.

## Features
- Cardiac risk analysis
- Time intelligence analysis
- Patient recovery trends
- Geographic healthcare insights
- Interactive Power BI dashboard

## Tools Used
- Power BI
- DAX
- Power Query
- Excel/CSV Dataset
